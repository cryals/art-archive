> **"Truth is the first casualty of the Frontier."**

**ART//ARCHIVE** is a high-fidelity immersive digital dossier system simulating a secure **NanoTrasen Directorate Operating System**. It serves as a classified archive for viewing character profiles, tactical evidence, and intercepted logs with a premium, diegetic user interface.

The project demonstrates advanced **React/Next.js** capabilities, including complex layout animations, glassmorphism interfaces, interactive sound design, and a curated "retro-futuristic" aesthetic.

## ðŸš€ Getting Started

Follow these instructions to deploy the secure archive on your local terminal.

### Requirements
*   Node.js 18.17 or later
*   npm or yarn

### Installation

1.  **Clone Repository**
    ```bash
    git clone https://github.com/your-username/art-archive.git
    cd art-archive
    ```

2.  **Install Dependencies**
    ```bash
    npm install
    # or
    yarn install
    ```

3.  **Initialize System**
    ```bash
    npm run dev
    ```

4.  **Access Protocol**
    Open the secure browser channel at `http://localhost:3000`.
